Vanilla JS task

  1. Enter your text and apply it
  2. Select any letter(-s) below:
    • by selecting an area with a range of letters and single clicking on one of them (works well only in Chrome now)
    • by holding down the control key and single clicking on the required letter(-s)
  3. Move cursor with letter(-s) attached, double click to set a new location
  4. Repeat if necessary
